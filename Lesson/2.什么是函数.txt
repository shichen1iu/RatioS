2.什么是函数???
    
    函数非常重要,许多的东西都是函数,

    比如说一台小汽车,你往加油口,加入一点汽油,那么它可以很好的工作
    你往里面加点可乐,那么它就会发生故障,

    比如说水果机,往里面放水果,得到果汁
    往里面放石头,它会坏掉.

    汽油 水果,这些东西本质都只是一堆数据
    (您可能会说,现实世界看得见摸的着,数据就只是几个数字呀,)
    (但问题是只要时间拉的足够长,现实世界也只是一堆数字..)

    也就是说,许多的东西都可以抽象成函数,它们可以搭配数据一起食用,
    那么我们先来制作一个函数
    函数有俩种,一种可以"返回",一种可以不"返回"
    
    void printSum(int x, int y) {
        std::cout << x + y << std::endl;
    }

    int Sum(int x,int y) {
        return x + y
    }

    function createHuman(name, age, sex) {
        return HumanTable.add(new Human(name, age, sex));
    }

    
    def SOLANA(program_id,accounts,ixD){
        ...
    }

    这些玩意,什么void int def 是啥玩意根本不重要对吗?
    什么std function 乱七八糟的,不重要呀.
    我们甚至不关心它到底是啥,反正就有返回和无返回.括号括起来的就是要塞入数据.
